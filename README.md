# Computer Vision
This repo contains Google Colab notebooks for ECSE 415

Assignment 1 focuses on low level image processing such as image de-noising, corner detection and edge detection.

Assignment 2 focuses on image featuring and matching techniques such as HAAR, SIFT, SURF features and PCA to recognize faces.

Assignment 3 focuses on Machine learning and Deep learning for image classification.

Assigment 4 focuses on image segmentation using techniques such as K-mean clustering. It also uses disparity map and optical flow to find image depth.
